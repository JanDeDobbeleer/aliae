## Example

```yaml
aliae:
  - alias: a
    value: aliae
    if: eq .Shell "zsh"
```

## Description

The `if` field is a stripped down go [text/template][go-text-template] conditional statement that
is evaluated with the current environment as the context. If the template statement
evaluates to true, the element is added to the shell.

Out of the box you get the following functionality:

| Name     | Type       | Description                                                 | Example                    |
| -------- | ---------- | ----------------------------------------------------------- | -------------------------- |
| `.Shell` | `string`   | the current shell name                                      | `eq .Shell "pwsh"`         |
| `.OS`    | `string`   | the current operating system (`windows`, `darwin`, `linux`) | `eq .OS "windows"`         |
| `env`    | `function` | retrieve an environment variable value                      | `eq (env "HELLO") "world"` |

[go-text-template]: https://golang.org/pkg/text/template/
